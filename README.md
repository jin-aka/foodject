# 파이널 프로젝트 : Foodject
#### <AI 플랫폼을 활용한 웹 서비스 개발>

## NCP 챗봇 기능을 활용한 Java기반의 반응형 배달 웹 서비스 개발

## 1. 팀 구성원 & 수행기간

- 팀장 : 문성용
- 팀원 : 배기진, 윤찬빈, 진가영
- 2022년 06월 28일 ~ 08월 18일
